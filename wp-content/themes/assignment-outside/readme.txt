== Description ==

Assignment Outside theme is only built for assignment tasks.

Theme template is used from BootstrapMade.
All images are used from google.

Pages includes homepage, events listing page, events details page, faq page, contact page & privacy policy page.

Assignment Outside theme is for lisiting the events.
All events are filtered and paginated using ajax.
Contact form 7 is used for contact form.
